# Аудит продукта «Живая память Обнинска»

Цель: **500 воспоминаний за 6 месяцев**. Критерий успеха экрана: «У меня тоже есть такая история — хочу добавить».

## Этап 1 (внедрено)
- CTA «У вас тоже есть такая история» на главной
- Кнопка «Добавить историю» в шапке и плавающая
- Реакции на StoryPage (6 типов + счётчики)
- Шеринг VK / Telegram / копирование
- Миграция `004_engagement.sql` (reactions, flags, popular_places)
- 40+ категорий в форме
- Админ: флаги «История дня», «На карте», «В архиве»
- Живая статистика в hero

## Этап 2 (внедрено, после 50 историй)
- Рейтинг популярных мест на карте
- Маршруты «Прогулки по памяти»
- ViralInvite на странице /share
- Значки ContributorBadges

## Этап 3–4
- Расширение маршрутов по районам при 200+ историях
- Лидерборд авторов при 500+ (подключить к Supabase)

Выполнить миграцию: `supabase db push` или SQL 001–004 в Dashboard.
